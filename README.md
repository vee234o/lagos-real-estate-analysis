# Lagos Real Estate Analysis

A data science project analyzing rental trends in Lagos, Nigeria. This project involves scraping real time property data, cleaning it, and performing exploratory data analysis (EDA) to understand pricing across different neighborhoods.

##  Project Structure
- `data/`: Contains raw and processed datasets.
- `notebooks/`: Jupyter notebooks for data cleaning, visualization, and analysis.
- `scraper/`: Python script used for scraping data from property listing websites.

##  Tools
- **Python**: Core programming language.
- **BeautifulSoup & Cloudscraper**: For scraping property data from *Nigeria Property Centre*.
- **Pandas & NumPy**: For data manipulation and cleaning.
- **Jupyter Notebook**: For interactive analysis.
